Exercício 1: Adicionar Elemento via Input
1,5 pontos
Objetivo: Crie uma função que adicione um novo item à lista quando o botão for clicado. O texto do item deve vir do input. Após adicionar, limpe o campo de input, ao lado do iten adicionado coloque um botão para remover o item.
    Exercício 2: Manipular CSS pelo DOM
    1,5 pontos
    Objetivo: Quando o botão for clicado, altere a cor de fundo do quadrado para "red", a largura para "200px" e a altura para "200px". Também adicione um border-radius de "10px". Ao clicar no botão novamente, o quadrado deve voltar à forma original (cor, tamanho e border-radius originais).
